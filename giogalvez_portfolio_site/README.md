# Giovanni Galvez Portfolio Site

This zip contains a complete static website:

- `index.html`
- `style.css`
- `app.js`
- `assets/README.txt`

## How to deploy

Upload all files to the web root of your host.

Typical Apache web root:
`/var/www/html/`

Typical static hosts:
Upload the contents of this folder, not the folder itself, unless your host expects a project folder.

## Optional files

The code references these optional files if you have them:

- `221.png`
- `new_space.png`
- `new_leaves.png`
- `Giovanni-Resume.pdf`

The site still runs without the images because the CSS has fallback gradient backgrounds.

If your resume file has another name, either rename it to:

`Giovanni-Resume.pdf`

or edit the resume download link in `index.html`.
